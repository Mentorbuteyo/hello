Introduction to Git

Below are some of the usefull commands in github

  • git clone (url): for downloading git ripository.
  • git add (filename): for adding new files.
  • git -m commit "message": for commiting changes.
  • git status: checking status.
  • git push: for uploading new files on the sever.
  • git pull: used when push has failed. or to pull all the changes made on the files online to your local machine.
  • git -am commit "message": shorthand for committing all changes.
